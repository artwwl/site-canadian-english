Sobre nós

Somos uma escola de inglês que traz a diversão dos acampamentos canadenses para a sala de aula.

Nossos professores aplicam uma abordagem comunicativa que proporciona aos estudantes uma forma mais natural de desenvolver a linguagem.

Desenvolvemos um programa pedagógico personalizado para cada estudante. Nossa avaliação de nível é feita pelo professor de forma progressiva. Ao longo do primeiro mês de aulas várias competências básicas serão testadas. Para garantir que o aprendizado tenha uma base sólida levamos em consideração aspectos diversos da comunicação. Com uma avaliação de nível de inglês personalizada podemos preparar aulas com conteúdos relevantes para os objetivos de cada aluno.
